Class Unit

A unit group in the game.

This may consist of a ship and any number of crew.

Methods

Unit:attack (tile, target)
Attacks either the 'crew' or 'ship' on a Tile in range.

Parameters:

  • tile Tile The Tile to attack.
  • target string Whether to attack 'crew' or 'ship'. Crew deal damage to crew and ships deal damage to ships. Consumes any remaining moves.

Returns:

    bool True if successfully attacked, false otherwise.
Unit:bury (amount)
Buries gold on this Unit's Tile. Gold must be a certain distance away for it to get interest (Game.minInterestDistance).

Parameters:

  • amount number How much gold this Unit should bury. Amounts <= 0 will bury as much as possible.

Returns:

    bool True if successfully buried, false otherwise.
Unit:deposit ([amount=0])
Puts gold into an adjacent Port. If that Port is the Player's port, the gold is added to that Player. If that Port is owned by merchants, it adds to that Port's investment.

Parameters:

  • amount number The amount of gold to deposit. Amounts <= 0 will deposit all the gold on this Unit. (default 0)

Returns:

    bool True if successfully deposited, false otherwise.
Unit:dig ([amount=0])
Digs up gold on this Unit's Tile.

Parameters:

  • amount number How much gold this Unit should take. Amounts <= 0 will dig up as much as possible. (default 0)

Returns:

    bool True if successfully dug up, false otherwise.
Unit:move (tile)
Moves this Unit from its current Tile to an adjacent Tile. If this Unit merges with another one, the other Unit will be destroyed and its tile will be set to nil. Make sure to check that your Unit's tile is not nil before doing things with it.

Parameters:

  • tile Tile The Tile this Unit should move to.

Returns:

    bool True if it moved, false otherwise.
Unit:rest ()
Regenerates this Unit's health. Must be used in range of a port.

Returns:

    bool True if successfully rested, false otherwise.
Unit:split (tile[, amount=1[, gold=0]])
Moves a number of crew from this Unit to the given Tile. This will consume a move from those crew.

Parameters:

  • tile Tile The Tile to move the crew to.
  • amount number The number of crew to move onto that Tile. Amount <= 0 will move all the crew to that Tile. (default 1)
  • gold number The amount of gold the crew should take with them. Gold < 0 will move all the gold to that Tile. (default 0)

Returns:

    bool True if successfully split, false otherwise.
Unit:withdraw ([amount=0])
Takes gold from the Player. You can only withdraw from your own Port.

Parameters:

  • amount number The amount of gold to withdraw. Amounts <= 0 will withdraw everything. (default 0)

Returns:

    bool True if successfully withdrawn, false otherwise.
