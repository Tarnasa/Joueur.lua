Class Game

Two player grid based game where each player tries to burn down the other player's buildings.

Let it burn.

Fields

self.name The name of this game, "{game_name}".
self.baseBribesPerTurn How many bribes players get at the beginning of their turn, not counting their burned down Buildings.
self.buildings All the buildings in the game.
self.currentForecast The current Forecast, which will be applied at the end of the turn.
self.currentPlayer The player whose turn it is currently.
self.currentTurn The current turn number, starting at 0 for the first player's turn.
self.forecasts All the forecasts in the game, indexed by turn number.
self.gameObjects A mapping of every game object's ID to the actual game object.
self.mapHeight The width of the entire map along the vertical (y) axis.
self.mapWidth The width of the entire map along the horizontal (x) axis.
self.maxFire The maximum amount of fire value for any Building.
self.maxForecastIntensity The maximum amount of intensity value for any Forecast.
self.maxTurns The maximum number of turns before the game will automatically end.
self.nextForecast The next Forecast, which will be applied at the end of your opponent's turn.
self.players List of all the players in the game.
self.session A unique identifier for the game instance that is being played.
